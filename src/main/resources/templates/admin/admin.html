Welcome to Admin Page

This is an admin page. This page consists of admin's detail information.
Instructions:

1. click 'Home' menu to come back to the admin page.

2. Click 'Create Task' menu to create a new task. You can write the task and select the user to whom you want to assign the task.

3. Click 'Tasks' menu to see the list of all tasks assigned to users. You can view individual tasks, check the submitted reply and send feedback.

4. Click 'Users' menu to see the list of all users registered into the program. You can view their info details.

5. Click dropdown and click 'Logout' to log out of the admin page back to home page.
