Welcome to Task Manager

This is an app in which an admin(i.e. manager, teacher, etc.) assigns tasks to the users(i.e. employees, students, etc.). The individual users can submit the tasks assigned to them. The admin can view the submitted answers and send feedback to the users.
Instructions:

1. Register as a new user (admin has been set by program).

2. The admin login email and password are given below for testing purpose. Use them to login into admin page.

3. New users can use email and password used for registering to login into user page.

4. The admin can create new tasks, view the tasks and users.

5. The users can view the tasks assigned to him/her by the admin, reply the tasks and submit it the to admin.
